# Start Consumer App

// turbo-all

1. Install dependencies if needed:
```bash
cd /home/rahul/Desktop/digimess/consumer && npm install
```

2. Start the Expo dev server:
```bash
cd /home/rahul/Desktop/digimess/consumer && npm start
```

The Expo dev server will start on port 8081. Press `a` for Android, `w` for web.
