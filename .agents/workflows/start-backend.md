# Start Backend

// turbo-all

1. Install dependencies if needed:
```bash
cd /home/rahul/Desktop/digimess/backend && npm install
```

2. Start the dev server with nodemon:
```bash
cd /home/rahul/Desktop/digimess/backend && npm run dev
```

The server will start on port 5000 with auto-restart on file changes.
