# Start Admin Panel

// turbo-all

1. Install dependencies if needed:
```bash
cd /home/rahul/Desktop/digimess/admin && npm install
```

2. Start the Vite dev server:
```bash
cd /home/rahul/Desktop/digimess/admin && npm run dev
```

The admin panel will start on http://localhost:5173 with hot module reloading.

Make sure the backend is running on port 5000 (use `/start-backend` workflow).
