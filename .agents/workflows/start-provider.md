# Start Provider App

// turbo-all

1. Install dependencies if needed:
```bash
cd /home/rahul/Desktop/digimess/provider && npm install
```

2. Start the Expo dev server:
```bash
cd /home/rahul/Desktop/digimess/provider && npm start
```

The Expo dev server will start on port 8082. Press `a` for Android, `w` for web.
